## Step 1: Get API Key (1 minute)

## Step 2: Add to Project (30 seconds)

Open `.env.local` (already open!) and replace:

```env
VITE_GROQ_API_KEY=your_groq_api_key_here
```

With your key:

```env
VITE_GROQ_API_KEY=gsk_abc123...
```

Save (Ctrl+S)

## 🐛 Issues?

**"Failed to get response"**
- Check key in `.env.local`
- Restart server
- Refresh browser (Ctrl+F5)

**"Invalid API key"**
- Get new key from https://console.groq.com/keys
- Copy entire key
- Replace in `.env.local`
