## ⚡ Quick Start (5 Minutes)

### 1️⃣ Get Your API Key (2 minutes)

**Go to**: https://platform.openai.com/api-keys

1. Click "Sign up" (or "Log in" if you have account)
2. **Easiest**: Click "Continue with Google"
3. Click "+ Create new secret key"
4. Name it: `Ubutaberahub Legal App`
5. Click "Create secret key"
6. **COPY THE KEY** (starts with `sk-proj-` or `sk-`)

---

### 2️⃣ Add Key to Project (1 minute)

1. Open file: `Ubutaberahub/.env.local`
2. Find this line:
   ```env
   VITE_OPENAI_API_KEY=your_openai_api_key_here
   ```
3. Replace with your key:
   ```env
   VITE_OPENAI_API_KEY=sk-proj-abc123...
   ```
4. Save (Ctrl+S)

---

### 3️⃣ Restart Server (1 minute)

**In terminal:**
```bash
# Stop server (Ctrl+C)
# Start again
npm run dev
```

**Or double-click:**
```
START_SERVER.bat
```

---

### 4️⃣ Test It! (1 minute)

1. Open: http://localhost:5173
2. Log in
3. Click "AI Assistant" in sidebar
4. Ask: "What are my rights if I'm arrested in Rwanda?"
5. **Get instant AI response!** 🎉

## 🐛 Troubleshooting

### Problem: "Failed to get AI response"

**Quick Fix:**
1. Check `.env.local` has your key
2. Key starts with `sk-` or `sk-proj-`
3. Restart server
4. Try again

### Problem: "Incorrect API key"

**Quick Fix:**
1. Go back to OpenAI platform
2. Create new API key
3. Copy entire key
4. Replace in `.env.local`
5. Restart server

### Problem: "Quota exceeded"

**Quick Fix:**
1. Go to: https://platform.openai.com/account/billing
2. Add payment method
3. Set spending limit ($5/month)
